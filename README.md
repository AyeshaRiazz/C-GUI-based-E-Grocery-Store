# 🛒 GUI E-Grocery Store (C++)

## 📖 Description

A **GUI-based E-Grocery Store application** developed in C++.
It provides an interactive interface for browsing products, managing a shopping cart, and generating bills.
This project combines **graphical user interface design** with core C++ programming concepts.

---

## 🚀 Features

* 🖥️ Interactive graphical interface
* 🛍️ Browse grocery items
* ➕ Add/remove items from cart
* 💰 Automatic bill calculation
* 🧾 Display final purchase summary

---

## 🛠️ Technologies Used

* **C++**
* GUI Library *(e.g., Qt / SFML / WinAPI — update if needed)*

---

## ▶️ How to Run

```bash id="b2k91d"
g++ main.cpp -o grocery_gui
./grocery_gui
```

*(Make sure required GUI libraries are installed)*
